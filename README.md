# Salary Converter

A single-file, zero-dependency web app that converts a salary figure across five time periods and four currencies simultaneously. Type into any cell — every other cell updates instantly.

## Features

- **Bidirectional, any-cell input** — enter a value in any period/currency combination; all 19 remaining cells recalculate on the fly.
- **Four currencies** — CZK, EUR, USD, GBP.
- **Five time periods** — Hourly, Daily, Weekly, Monthly, Yearly.
- **Live exchange rates** — fetched automatically on load from three independent public APIs, queried in parallel. The highest-priority source that succeeds is used; the others serve as automatic fallbacks.
- **Resilient offline mode** — if every rate source fails, the app falls back to hardcoded approximate rates and clearly flags this with a warning banner.
- **Rate status panel** — shows the active source, the rate date, the current EUR→CZK/USD/GBP values, and the per-source success/failure status at a glance.
- **Manual refresh** — a *Refresh rates* button re-queries all sources and updates calculations if a filled cell is present.
- **Flexible number input** — accepts both `.` and `,` as decimal separators; ignores spaces used as thousands separators.
- **Truncated output** — displayed values are cut (not rounded) to 2 decimal places, with thousands separated by spaces for readability.

## Time-period assumptions

| Period  | Hours assumed |
|---------|--------------|
| Hourly  | 1            |
| Daily   | 8            |
| Weekly  | 40 (5 days)  |
| Monthly | 172 (21.5 working days × 8 h) |
| Yearly  | 2 080 (260 working days × 8 h) |

## Exchange rate sources

Sources are queried in parallel on every load and refresh. Priority order (first successful response wins):

1. **Frankfurter (ECB)** — `api.frankfurter.app` — rates from the European Central Bank.
2. **open.er-api.com** — free tier of Exchange Rates API.
3. **Currency API via jsDelivr** — `@fawazahmed0/currency-api` CDN mirror.

All sources are free and require no API key.

## Usage

No build step, no dependencies, no server required.

```
# Clone or download the file, then open it directly in any modern browser:
open SalaryConverter.html
```

Or host it anywhere that serves static files — GitHub Pages, Netlify, a plain nginx server, etc.

## Browser support

Any modern browser with `fetch` and ES2017+ support (Chrome 66+, Firefox 60+, Safari 12+, Edge 79+).

## File structure

Everything lives in a single `SalaryConverter.html` file:

- **CSS** — custom-property-based dark theme, responsive table, status panel styles.
- **HTML** — minimal shell; the grid rows are generated by JavaScript.
- **JavaScript** — grid builder, calculation engine, FX fetch/fallback logic, UI rendering.

## Calculation logic

All values are converted through a common intermediate unit: **EUR per hour**.

```
eurPerHour = inputAmount / periodHours / rates[inputCurrency]
outputValue = eurPerHour × periodHours × rates[outputCurrency]
```

This means any single cell fully determines the rest of the grid, and switching the source cell always produces consistent results.
